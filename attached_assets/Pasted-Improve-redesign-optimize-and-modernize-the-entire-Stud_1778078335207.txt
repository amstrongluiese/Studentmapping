SCHOOL IMPORT WORKFLOW

When admin uploads a file:

1. Read uploaded rows
2. Extract school data
3. Normalize school names
4. Remove duplicates
5. Validate records
6. Detect missing coordinates
7. Auto-geolocate missing coordinates using Nominatim
8. Save all processed schools permanently
9. Automatically generate map markers
10. Update analytics dynamically

REQUIRED IMPORT FILE STRUCTURE

The Excel upload must support these columns:

| School Name | Municipality | Institution Type | Latitude | Longitude | Altitude |

Example:
| Lumban National High School | Lumban | Public High School | 14.2971 | 121.4590 | 12 |

COORDINATE HANDLING

If coordinates already exist:
- instantly create markers
- skip API calls
- render immediately

If coordinates are missing:
System should:
1. use Nominatim API
2. search school location
3. fetch latitude/longitude
4. auto-fill missing data
5. save permanently

IMPORTANT:
Save all verified coordinates permanently.

Do NOT repeatedly call APIs for already verified schools.
